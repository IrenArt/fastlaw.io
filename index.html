Примеры задач

ПО под ключ

«Роспечать»: автоматический секретарь для работы с письмами

Задача

На почту “Роспечати” приходят тысячи писем в день.
Чтобы посмотреть их, аннотировать и передать ответственному лицу, несколько человек тратят весь день.
Задача: автоматизировать этот процесс, снять нагрузку
с сотрудников.

Решение

ПО выполняет задачи, приблизительно равные человеку.
Теперь вместо 3-х сотрудников нужен 1, который контролирует результаты ПО.

ПО под ключ

«Правокард»:
сайт с диалоговым интерфейсом

Задача

«Правокард» создал базу знаний, где собрана вся актуальная информация о налогах для самозанятых.
Нужна была команда, которая напишет сайт, сделает его интерфейс простым и удобным.

Решение

На ¾ сократился поток заявок на юристов, система работает как фильтр лидов, фильтруя людей с простыми и неденежными вопросами. Диалоговый интерфейс понимает вопросы на естественном языке и может дообучаться.

ПО под ключ

«Правокард»: ПО, которое извлекает
данные из фотографий документов

Задача

Упростить и ускорить заполнение форм на web-сайтах и в мобильных приложениях «Правокарда», требующих ввода персональных данных.

Решение

Разработали автоматизарованный онбординг клиента. Технология распознает данные фотографий с документов и заполняет регистрационную форму. Так, время на подачу документа в налоговую уменьшается в 10 раз.

Консалтинг

«Mediascope»:
ПО, которое анализирует рекламу

Задача

«Mediascope» занимается маркетинговыми исследованиями. Изначальной задачей было заменить операторов. Чтобы составить ТЗ мы изучили как они работают: вручную вбивают тематику рекламы, рекламодателя и марку товара для поиска.

Решение

ПО автоматизирует работу оператора, анализирует рекламу и отправляет данные заказчику. Производительность оператора 1-1,5 единиц контента в день, а ПО показывает результат в 200 раз больше.

ПО под ключ

«Lobster.Media»:
поисковый движок для изображений

Задача

«Lobster.Media» — поисковик по картинкам. Он позволяет пользователям соцсетей продавать свой контент, а агентствам получать его понизким ценам. Компания обратилась к нам, чтобы разработать поисковый движок.

Решение

ПО ищет изображение на основе его контента, а не метаинформации. Мы быстро внедрили сервис и продолжили работать с компанией на постоянной основе, создавая ML сервисы для них.

Консалтинг
MVP

«РЖД»: чат-бот, который
принимает звонки машиниста

Задача

Автоматизировать работу оператора «РЖД», котрый принимает звонки от машинистов в случае нарушения на путях и станциях.
