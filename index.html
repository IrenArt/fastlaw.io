ПО под ключ

«Роспечать»: автоматический секретарь для работы с письмами

Задача

На почту “Роспечати” приходят тысячи писем в день.Чтобы посмотреть их, аннотировать и передать ответственному лицу, несколько человек тратят весь день. Задача: автоматизировать этот процесс, снять нагрузкус сотрудников.

Решение

ПО выполняет задачи, приблизительно равные человеку. Теперь вместо 3-х сотрудников нужен 1, который контролирует результаты ПО.
